Step 1: Install Pygame

Before you can start using Pygame, you need to install it. Open your terminal (command prompt on Windows) and run the following command to install Pygame using pip, Python's package manager:

pip install pygame

Step 2: Import Pygame

Create a Python script (e.g., game.py) and open it in your favorite code editor. Start by importing the Pygame library:

import pygame

Step 3: Initialize Pygame

Before you can use Pygame features, you need to initialize it:

pygame.init()

Step 6: Draw a Shape

Now, let's draw a simple shape, like a red rectangle, on the screen. Add the following code inside the game loop:

// Add game logic here
screen.fill((0, 0, 0))  # Fill the screen with black

# Draw a red rectangle (x, y, width, height)
pygame.draw.rect(screen, (255, 0, 0), (100, 100, 50, 50))

pygame.display.flip()

Step 7: Quit Pygame

Finally, don't forget to quit Pygame when you're done:

pygame.quit()
